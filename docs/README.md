# AZ Learner — The Squad Companion

> **Print, Chat, Grind, Shine.** The ultimate utility tool for the academic weapon.

Live site: [azlearner.me](https://azlearner.me)

## About

AZ Learner is a student-focused web platform designed to simplify campus life. It brings together printing, academic tools, campus events, and community features in one place — giving students a "Digital Exhale" from the grind.

## Features

- **Print & Deliver 🖨️** — Submit documents for printing and delivery right on campus.
- **The Arena** — A competitive study and gaming space for students.
- **Course Circles** — Connect with classmates around shared courses.
- **Campus Events** — Stay up to date with events happening around you.
- **GPA Calculator** — Quickly calculate and track your GPA with the dedicated [AZ Academic Engine](/AZ-GPA-Calculator/).
- **The Vault** — Access shared academic resources.
- **CV Builder** — Build and download a professional CV or résumé.
- **AZ Labs** — Career acceleration and academic optimisation tools.
- **The AZ Podcast** — Podcast series exploring Gen Z culture, philosophy, and growth.
- **Reviews** — Student reviews and ratings for the platform.
- **Wallpapers** — 105+ free stunning wallpapers for students.
- **Campus Stickers** — AZ Learner branded sticker packs.
- **Ambassador Program** — Campus ambassador portal with courses, challenges, and leaderboard.
- **Surveys** — Community research surveys for platform improvement.

## Ambassador Program

The AZ Ambassador Program lets students represent AZ Learner on their campus. Ambassadors promote the platform, organize events, provide feedback, and support fellow students — while gaining leadership skills, networking opportunities, and exclusive access to new features.

The ambassador portal (`ambassador.html`) includes:
- Training courses with module-by-module content
- Monthly challenges with task tracking and badges
- Achievement points leaderboard with peer recognition
- Program announcements and sign-up management

## Tech Stack

- HTML, CSS (Tailwind CSS via CDN)
- Vanilla JavaScript (ES Modules)
- Firebase (Firestore + Authentication)
- Paystack (payments)
- Google Fonts & Material Symbols / Font Awesome
- GitHub Pages (hosting)
